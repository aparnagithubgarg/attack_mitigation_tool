# Attack Mitigation Tool

An interactive web application designed to demonstrate common web security vulnerabilities and their mitigation techniques. Built for educational purposes to understand how attacks work and how to prevent them.

## Features

- **Interactive Vulnerability Demos**: Hands-on examples of critical web security flaws
- **Real-time Attack/Defense Toggle**: Switch between vulnerable and secure modes
- **Educational Content**: Clear explanations of vulnerabilities and mitigation strategies
- **Responsive Design**: Works seamlessly across all devices

## Vulnerabilities Covered

| Attack Type | Vulnerability | Mitigation |
|-------------|---------------|------------|
| **SQL Injection** | Unsafe string concatenation in queries | Parameterized queries with placeholders |
| **Cross-Site Scripting (XSS)** | Direct HTML rendering of user input | Output encoding with HTML escaping |

## Tech Stack

**Frontend:**
- HTML5, CSS3, JavaScript
- Responsive design with modern UI

**Backend:**
- Python 3.x
- Flask web framework
- SQLite database

## How to Use
**For Learners:**
- Browse Modules - Navigate through different vulnerability sections

- Try Attacks - Use provided payloads to test vulnerabilities

- Toggle Security - Switch to see how fixes prevent attacks

- Read Documentation - Understand the theory behind each attack

**For Educators:**
- Demo Ready - Use in classrooms or workshops

- Customizable - Easy to extend with new content

- Safe Environment - All attacks are contained and educational
